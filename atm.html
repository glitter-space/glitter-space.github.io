Bank Of America ATM User Interface Design

I was tasked with reworking Bank of America’s ATM user interface to increase the safety of the user by reducing the amount of time spent at the machine.

Information Architecture

The current interface allows for many transactions and each one requires several steps. Certain action, such as changing the user’s PIN, would be easier and safer to change through a website or mobile app.

Visual Design

The user experiece can be further streamlined through visual changes to the ATM interface. Larger buttons, saved preferences, and a toggle view for balances all contribute to a more user friendly interface.

The above work was designed by me as a student at the Savannah College of Art and Design
