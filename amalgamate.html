Amalgamate Website Redesign

Web Design, Digital, UX

The Problem

Amalgamate, a charity-focused dance company, needs a website redesign. The current design uses an outdated visual identity and is difficult to navigate.

The Goal

The new design should improve the navigation and apply the updated visual identity of the company. The website should be built with a website builder so that it can be easily updated by company members in the future.
